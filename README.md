# 📦 Black Box Master - Ultimate Suite

![License](https://img.shields.io/badge/license-GPLv3-purple.svg)
![Version](https://img.shields.io/badge/version-1.0.0--beta-neon)

**Black Box Master** هو مشروع مفتوح المصدر مصمم لتبسيط عملية تحويل ملفات Python إلى تطبيقات Windows (EXE) مستقلة، مع مجموعة من الأدوات المتقدمة لإدارة النظام وتشفير البيانات.

## 🚀 المميزات الرئيسية
* **Python to EXE:** تحويل ذكي وسريع باستخدام واجهة رسومية احترافية.
* **Inno Setup Integration:** توزيع البرنامج كملف تثبيت (Installer) رسمي.
* **Open Source:** الكود بالكامل متاح للمراجعة والتطوير تحت رخصة GPLv3.
* **Hacker Aesthetic:** واجهة مستخدم نيون جذابة مستوحاة من بيئات التطوير المتقدمة.

## 🛠️ المتطلبات
* نظام تشغيل Windows 10/11.
* تثبيت Inno Setup (إذا كنت ترغب في بناء ملف التثبيت بنفسك).
